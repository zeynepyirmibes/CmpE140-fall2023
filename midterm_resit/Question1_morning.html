Question Description

Assume that you are running a grocery store and need to keep track of your stock through a function named stock_situation(gro_stock).
The function takes a named vector as input to report the vegetable with the least amount and the average weight (in kg) of scarce veggies. (Tip: There may be test cases with no scarce veggies).
A vegetable is considered scarce if the stock has less than or equal to 30 kg of that item.
Write the stock_situation function to find the vegetable with the least amount and calculate the average weight of scarce vegetables in the input named vector.
Input
A named vector: where names represent the vegetable names (strings), and values represent the corresponding stock amounts in kg (numeric).
The vector may contain various vegetables and their respective stock amounts. Names are not limited to the ones given in test cases.
Output
A named vector containing:
  • The name of the vegetable with the least amount (string) as "Least_Vegetable"
  • The average weight (in kg) of scarce vegetables (numeric), rounded to two decimal places, as "Average_Weight"
Examples
stock1 <- c(carrots = 25.5, broccoli = 40.2, spinach = 10.7, tomatoes = 55.1, cucumbers = 8.9)
stock_situation(stock1)

##Least_Vegetable  Average_Weight 
##    "cucumbers"         "15.03" 


stock2 <- c(apples = 40.0, bananas = 35.2, oranges = 50.3, peaches = 25.1, cherries = 60.2)
stock_situation(stock2)

##Least_Vegetable  Average_Weight 
##      "peaches"          "25.1" 


stock3 <- c(apples = 12.3, bananas = 5.8, oranges = 35.2, peaches = 40.0, cherries = 10.5)
stock_situation(stock3)

##Least_Vegetable  Average_Weight 
##      "bananas"          "9.53"
