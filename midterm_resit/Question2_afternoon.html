Question Description

The Air Quality Index (AQI) is a measurement of particulate matter in the air, and shows how polluted the air is. Below is a vector that contains the AQI values (sorted in an ascending way) for some cities in Turkey: 
aqi_2022 <- c(12, 25, 47, 56, 65, 76, 89)
Assume that the vector will always have an odd number of values. Write a function called middle_avg_difference(vec) that finds the AQI value in the middle of the vector, and the average of the values.
  • If the middle value is larger than the average of the vector, calculate, round (to two decimal places) and return: middle - average
  • If the middle value is smaller than the average of the vector, calculate, round (to two decimal places) and return: average - middle
  • If the middle value is equal to the average of the vector, return the character value: "same"

Example 1:

aqi_2022 <- c(12, 25, 47, 56, 65, 76, 89)
The middle value in aqi_2022 is 56, and the average is 52.86. Then the output is, middle - average = 56 - 52.86 = 3.14 
middle_avg_difference(aqi_2022)
3.14

Example 2:

aqi_2021 <- c(12, 13, 14)
The middle value in aqi_2021 is 13, and the average is 13. Then the output is, the character value "same" 
middle_avg_difference(aqi_2021)
'same'

Example 3:

aqi_2023 <- c(12, 25, 47, 56, 65, 76, 89, 100, 120)
The middle value in aqi_2023 is 65, and the average is 65.56. Then the output is, average - middle = 65.56 - 65 = 0.56 
middle_avg_difference(aqi_2023)
0.56
