Question Description

Write a function in R named lucas that takes a non-negative integer 'n' as input and returns the nth number in the Lucas series.
The Lucas series starts with the numbers 2 and 1 for n = 0 and n = 1, respectively.
Each subsequent number is the sum of the two preceding numbers for n > 1.
  • L0 = 2
  • L1 = 1
  • Ln = Ln-1 + Ln-2
Examples

lucas(5) is 7
lucas(23) is 39603
lucas(0) is 2
lucas(1) is 1
lucas(235) is 8.00032847516874e+48
