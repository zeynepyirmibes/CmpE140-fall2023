Question Description

Write a function in R called vending_machine that takes a coin vector and an item name as inputs, and returns the remaining change as another coin vector.
The vending machine stores five items and accepts coins of 5, 10, and 25 cents i.e. nickel, dime, and quarter, respectively.

Item Price ($)
Chips 1.30
Chocolate 1.00
Coke 1.60
Soda 0.80
Water 0.50

Example of named coin vector: 
coins <- c(nickels = 3, dimes = 1, quarters = 5)
 
 nickels    dimes quarters 
       3        1        5
If the user does not have enough money to buy the item, return "Not enough money" instead of remaining coin vector.
The "machine" should return same named vector of coins as the input.
When calculating the change vector, the vending machine first attempts to give quarters, then dimes, and finally nickels, regardless of what the customer inserted in the first place.
You can assume that the machine always has sufficient coins to return.

Note that $1 is 100 cents.

Examples

vending_machine(c(nickels=0, dimes=1, quarters=6), "Coke")

 nickels    dimes quarters 
       0        0        0 


vending_machine(c(nickels=0, dimes=2, quarters=5), "Chocolate")

 nickels    dimes quarters 
       0        2        1 


vending_machine(c(nickels=2, dimes=2, quarters=1), "Chips")

"Not enough money"


vending_machine(c(nickels=2, dimes=2, quarters=4), "Soda")

 nickels    dimes quarters 
       0        0        2 


vending_machine(c(nickels=12, dimes=3, quarters=5), "Soda")
 
 nickels    dimes quarters 
       0        1        5 


vending_machine(c(nickels=40, dimes=0, quarters=0), "Chocolate")

 nickels    dimes quarters 
       0        0        4
