Question Description

As the IT administrator of your company, you need to write a function named secure_password that takes a password as input and determines its strength based on the following criteria:
  • It should have a length longer than 8 characters (length > 8).
  • It should contain at least one lowercase letter, one uppercase letter, one number, and one special symbol from the following set {!, #, + , %, &, *,-}.
For each condition met, the function increments the security class according to the following:
  • Length > 8: 1 point
  • Contains lowercase, uppercase, number, and symbol: 1 point each
After evaluating the conditions, the function should report the strength of the password according to the following table:
Point Strength
5 Very Strong
4 Strong
3 Medium
2 Weak
1 Very Weak

Examples

secure_password("123")
"Very Weak"

secure_password("weakerpass")
"Weak"

secure_password("PassoPass")
"Medium"

secure_password("StrongPassword123")
"Strong"

secure_password("V3ry5trongPassw*rd!23")
"Very Strong"
